외부 도구 확장을 위해 MCP 서버를 연결하고, coreline-agent에서 MCP 기반 tool inventory를 사용할 수 있는 최소 경로를 만든다.

## 개발 범위
- MCP client/connection manager 최소 구현
- MCP 서버 설정 로드
- MCP tool inventory 노출
- coreline-agent tool bridge 초안
- 테스트 및 최소 문서 추가

## 제외 범위
- MCP marketplace
- remote plugin 배포 시스템
- 완전한 MCP 권한/채널 관리 고도화
- proxy hosted tools와의 완전 통합
