이 문서는 현재 `coreline-agent`에서 로컬 테스트는 통과하지만 실제 외부 연동 smoke가 부분적으로 닫히지 않은 상태를 정리하고, cloud/provider/MCP/proxy 실환경 기준의 검증을 마무리하기 위한 작업 문서다.

## 개발 목적
- BLOCKED 상태로 남아 있는 외부 smoke를 가능한 범위까지 실제로 닫는다.
- 로컬 smoke와 cloud smoke의 결과를 명확히 구분한다.
- 문서/체크리스트/CI 기준에 실제 검증 결과를 반영한다.

## 개발 범위
- Anthropic / OpenAI / Gemini 실제 smoke 재실행
- OpenAI-compatible / local smoke 재확인
- MCP 실제 서버 1~2개 smoke 재확인
- proxy / hosted tools 실제 요청 smoke 재확인
- BLOCKED / OUT-OF-SCOPE / PASS 기준 정리
- 관련 문서/체크리스트 반영

## 제외 범위
- 신규 provider 추가
- smoke 하니스를 새 아키텍처로 재작성
- proxy 기능 확장
- planner 알고리즘 변경
- MCP remote transport 추가
