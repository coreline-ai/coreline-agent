## 개발 목적
현재 plain text만 표시하는 TUI streaming/output 영역에 최소 markdown/code fence 렌더링을 추가해 가독성을 개선한다.

## 개발 범위
- `src/tui/streaming-output.tsx` 최소 markdown 렌더링
- 코드 fence 감지 및 block 스타일 표시
- inline code/strong 정도의 최소 강조
- `tool-result.tsx`와 줄바꿈/폭 호환성 보강
- TUI 관련 unit/helper 테스트 추가

## 제외 범위
- 완전한 markdown 엔진 도입
- syntax highlighting 라이브러리 추가
- 클릭 가능한 링크/이미지 렌더링
- 복잡한 diff viewer
